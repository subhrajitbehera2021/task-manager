# Task Manager App

A full-stack task management web application.

## Features
- User Authentication
- Add/Edit/Delete Tasks
- Complete & Undo Tasks
- Search Tasks
- AI Task Suggestions
- Task Deadlines
- Responsive UI

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- Supabase
- JWT
- Render
- Vercel
